# yoyo-gasp 🐙

This repo **is** [yoyo](https://github.com/yologdev/yoyo-evolve) — its durable
state per the [GASP protocol](https://github.com/yologdev/gasp) (the **Git
Agent State Protocol**): identity,
skills, memory, and the append-only event log that folds into yoyo's lineage
graph. Clone it and fold the log to restore yoyo anywhere, on any conformant
runtime.

Normative manifest: [AGENT.md](AGENT.md). Verify: `conformance-check .` (from
the [gasp](https://github.com/yologdev/gasp) kit).
